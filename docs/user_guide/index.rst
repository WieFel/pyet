User Guide
==========
Here you can find guidance on how to get started with pyet, from installing the package on your computer to selecting
a method and estimating potential evapotranspiration. A special section is provided on notation and units, to guide
users to use the correct input data for the different methods.

GitHub Discussions
------------------
If you have any questions on how to use *pyet* not answered in the documentation, please ask your question on the
`GitHub Discussions forum <https://github.com/phydrus/pyet/discussions>`_.
